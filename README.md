# fancy-todo
Fancy Todo App is a database for todos

** RESTful Endpoints **
----
- `POST /todos`
- `GET /todos`
- `GET /todos/:id`
- `PUT /todos/:id`
- `PATCH /todos/:id`
- `DELETE /todos/:id`

** POST Todo **
----
    Write a new Todo data to database

** URL **
----
`/todos/`

** Method: **
----
